Берем готовую efficientnet, предобученную на 1.2M изображений ImageNet. Я пробоаола B0 и B1, 
разные гиперпараметры и разное количество эпох. Оптимизатор - AdamW(просто я долбанула какой-то крутой).
Многое тоже не особо обдуманно выбрано. Ни времени, ни ресурсов не хватит, чтобы всё протестировать за 3 дня.
Лучший результат дал B0.

Имеется:

#1 Валидация модели на части обучающей выборки

#2 Автоматическое сохранение модели при обучении

#3 Вывод различных показателей в процессе обучения (например, значение функции потерь на каждой эпохе)

#4 Построение графиков, визуализирующих процесс обучения (график зависимости функции потерь от номера эпохи обучения, и т.п.)

#5 Построение матрицы ошибок, оценивание чувствительности и специфичности модели

#6 Использование аугментации и других способов синтетического расширения набора данных 
(дополнительным плюсом будет обоснование необходимости и обоснование выбора конкретных типов аугментации)
Гистологические образцы очень вариативны, модель должна понимать, что крути-верти, меняй яркость, обрезай, это
все равно тот же самый образец. Мы добавляем ещё данных, боремся с переобучением.
